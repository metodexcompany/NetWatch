# NetWatch

**Кто стучится к тебе прямо сейчас?**

NetWatch — монитор сетевых соединений для Windows. Показывает какие программы отправляют данные, куда, и позволяет заблокировать подозрительные подключения одним кликом.

![.NET](https://img.shields.io/badge/.NET-8.0-purple?logo=dotnet)

## Возможности

- **Мониторинг в реальном времени** — все TCP-соединения с именами процессов
- **Классификация трафика** — автоматическое определение известных сервисов (Google, Yandex, Cloudflare и 50+ других)
- **GeoIP** — страна и организация для каждого IP-адреса
- **Проверка подписи** — цифровая подпись процессов через Windows Authenticode
- **Блокировка IP** — один клик, правило Windows Firewall
- **Алерты** — уведомления при обнаружении подозрительных соединений
- **Скорость сети** — download/upload в реальном времени
- **Настройки** — интервал обновления, GeoIP вкл/выкл, сохранение между сессиями

## Установка

### Установщик
Скачайте `NetWatch_Setup_1.0.0.exe` из [Releases](../../releases) и запустите.

### Из исходников
```bash
cd NetWatch
dotnet publish -c Release -r win-x64 --self-contained -o ../build
```

## Требования

- Windows 10/11 (x64)
- Права администратора (для чтения TCP-соединений и управления файрволом)

## Технический стек

- C# / .NET 8 / WPF
- WMI (MSFT_NetTCPConnection) для сетевых соединений
- WinVerifyTrust API для проверки подписей
- ip-api.com для GeoIP (опционально, можно отключить)
- netsh advfirewall для блокировки IP
- SQLite для истории (в разработке)

## Как это работает

1. Каждые N секунд опрашивает TCP-соединения через WMI
2. Определяет имя процесса и путь к exe по PID
3. Проверяет цифровую подпись через Windows API
4. Определяет страну/организацию через GeoIP API
5. Классифицирует: безопасно / неизвестно / угроза
6. Показывает в дашборде с возможностью заблокировать

## Безопасность

- Все данные хранятся локально
- GeoIP запросы можно отключить в настройках
- Открытый исходный код — проверяйте сами
